# Movianx First Cinematic Reinterpretation Sequence

## Sequence

`Night of the Living Dead` (1968) — Barbara dissociation reinterpretation.

## Why This Node

This is the safest first cinematic proof-of-concept because it is emotionally powerful without requiring a high-motion generation pass. Barbara's farmhouse shock can be deepened through atmosphere, shadow, silence, and dissociation while the original scene remains visibly underneath.

## Deliverables In This Package

- `node.json` — selected reinterpretation node
- `prompt-package.json` — provider-ready prompt and source clip specification
- `comfyui-atmosphere-shift-filled.json` — filled atmosphere-shift workflow
- `assembly-plan.json` — reintegration and side-by-side assembly plan
- `continuity-analysis.md` — validation criteria and emotional realism assessment

## Render Status

The sequence is packaged for external video-to-video generation.

The final rendered clip is not present in this local repo context because required rendering components are unavailable here:

- no checked-in original film clip
- no ffmpeg binary
- no running ComfyUI video service
- no Kling or Runway credentials/runner

## Intended Outputs

- `output/barbara-dissociation-reinterpretation.webm`
- `output/barbara-dissociation-timeline-poc.webm`
- `output/barbara-dissociation-side-by-side.webm`

## Creative Bar

The reinterpretation succeeds only if the audience feels the original film is still alive underneath the generated pass.

This should feel like a hidden emotional version of the scene, not a replacement scene.
